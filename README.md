# AI Куратор — Telegram бот

## Как запустить за 10 минут

### Шаг 1 — Создай бота в Telegram
1. Открой @BotFather в Telegram
2. Напиши `/newbot`
3. Придумай имя и username боту
4. Скопируй **BOT_TOKEN** который он выдаст

### Шаг 2 — Узнай свой Telegram ID
1. Напиши боту @userinfobot
2. Он пришлёт твой **ID** — запомни его

### Шаг 3 — Добавь бота в группу
1. Открой свою группу с учениками
2. Добавь бота как участника
3. Дай боту права **читать сообщения** и **писать сообщения**
4. Важно: в настройках группы → Privacy mode бота → выключить (через BotFather командой `/setprivacy` → выбрать бота → `Disable`)

### Шаг 4 — Задеплой на Railway
1. Зарегистрируйся на [railway.app](https://railway.app)
2. Нажми **New Project → Deploy from GitHub repo**
   - Либо: **New Project → Empty project → Add service → GitHub repo**
3. Загрузи эти файлы в репозиторий GitHub (или используй Railway CLI)
4. В настройках проекта добавь переменные окружения:

```
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_TELEGRAM_ID=твой_telegram_id
ANTHROPIC_API_KEY=sk-ant-...
AUTO_THRESHOLD=10
```

5. Railway сам установит зависимости и запустит бота

## Как пользоваться

### В личке с ботом (команды для тебя):
- `/stats` — статистика и все правила в памяти
- `/manual` — переключить в ручной режим (все ответы через тебя)
- `/auto` — переключить в авто-режим (бот отвечает сам)

### Процесс работы:
1. Ученик пишет в группу
2. Бот пишет тебе в личку с черновиком ответа
3. Ты нажимаешь **✅ Одобрить** или **✏️ Исправить**
4. При исправлении — пишешь свой вариант следующим сообщением
5. Бот автоматически учится на твоих правках

### Авто-режим:
После 10 проверок с точностью ≥80% бот сам переключается в авто-режим.
Ты всё равно получаешь уведомления о каждом ответе, но без кнопок.
Принудительно переключить: `/manual` или `/auto`
